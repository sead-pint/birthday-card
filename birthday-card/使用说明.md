# 🎂 交互式生日贺卡使用指南

这份代码是一个运行在网页上的交互式生日贺卡。你可以把它部署到网上，生成一个链接发给你的女朋友。

## 📁 文件结构
- `index.html`: 主页面结构
- `css/style.css`: 样式文件（颜色、动画）
- `js/script.js`: 交互逻辑（点击事件、场景切换）

## 🛠️ 如何修改定制

### 1. 修改背景音乐 🎵
- 找一首她喜欢的歌，重命名为 `bgm.mp3`。
- 将文件放入 `birthday-card` 文件夹内（与 `index.html` 同级）。
- 如果文件名不是 `bgm.mp3`，请在 `index.html` 第 12 行修改：
  ```html
  <source src="你的音乐文件名.mp3" type="audio/mpeg">
  ```

### 2. 添加照片 📷
- 找一张好看的照片，放入 `birthday-card` 文件夹。
- 打开 `index.html`，找到第 63-68 行：
  ```html
  <!-- 这里可以放照片 -->
  <div class="photo-frame">
      <div class="photo-placeholder">
          <!-- 修改下面这行，去掉注释，把 src 改成你的照片文件名 -->
           <img src="你的照片文件名.jpg" alt="Photo"> 
      </div>
  </div>
  ```
- *注意：* 你可能需要调整 CSS 中的 `.photo-frame`样式来适应照片比例。

### 3. 修改祝福语 ✍️
- 打开 `index.html`，修改第 72-77 行的文字内容。

## 🚀 如何在手机上查看（非常重要）

微信不能直接发送 HTML 文件并在手机上完美运行。最好的方法是**将网页发布到互联网上**，然后发链接给她。

### 方法一：使用 GitHub Pages (推荐，免费且永久)
1. 在 GitHub 上创建一个新仓库（Repository）。
2. 将这三个文件/文件夹上传到仓库。
3. 进入仓库的 `Settings` -> `Pages`。
4. 在 `Branch` 选择 `main` (或 master)，点击 `Save`。
5. 等待几分钟，你会获得一个链接（例如 `https://你的用户名.github.io/仓库名/`）。
6. **把这个链接发给微信发给她！**

### 方法二：本地测试 (仅自己电脑看)
1. 在 VS Code 中安装 "Live Server" 插件。
2. 右键点击 `index.html`，选择 "Open with Live Server"。
3. 浏览器会自动打开预览。

## 💡 小贴士
- 很多手机浏览器（包括微信内置浏览器）禁止自动播放音乐，所以在这个设计中，我添加了一个“点击信封”的动作。只有用户点击了屏幕，音乐才会开始播放，这是浏览器的安全策略。
- 建议先在自己的手机上打开链接测试一下效果，确认文字没有错位。
